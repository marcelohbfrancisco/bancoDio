Criando um Banco Digital com Java e Orientação a Objetos
Descrição
Este repositório contém o projeto do laboratório "Criando um Banco Digital com Java e Orientação a Objetos". O projeto simula a operação básica de um sistema bancário digital, onde são implementados os conceitos fundamentais da programação orientada a objetos (POO) e o ecossistema Java.

Funcionalidades
Criação de diferentes tipos de contas bancárias (Conta Corrente, Conta Poupança, Conta Jovem e Conta Salário).
Operações de depósito, saque e transferência entre contas.
Solicitação e autorização de empréstimos para contas elegíveis.
Geração de extrato bancário com informações detalhadas da conta.
Conceitos Abordados
Encapsulamento
Protege o estado interno do objeto e esconde a implementação detalhada dos métodos.

Herança
Permite a criação de novas classes com base em classes existentes para promover a reutilização de código e criar uma hierarquia de classes.

Polimorfismo
Permite que objetos sejam tratados como instâncias de suas classes pai, promovendo flexibilidade e integração entre diferentes tipos de contas bancárias.

Classes Abstratas e Interfaces
Usadas para definir contratos para classes e para separar as operações bancárias da lógica de implementação das contas.

Métodos Estáticos
Utilizados nas classes utilitárias para realizar operações bancárias que não dependem do estado de um objeto específico.
